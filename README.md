# 🏥 Hospital Management System (GUI)

A desktop-based hospital management system developed using Python Tkinter to manage patient records, doctor details, and appointments efficiently.

## 🚀 Features

- Patient registration and record management

- Doctor management system

- Appointment scheduling

- Data storage using text files

- Simple and user-friendly GUI interface

## 💻 Technologies Used

- Python

- Tkinter (GUI)

## 📁 Project Files

- hospital_gui.py → Main GUI application

- doctor_manager.py → Handles doctor-related operations

- doctor.txt → Stores doctor data

- patients.txt → Stores patient data

- about.txt → Additional information

## ▶️ How to Run

Clone the repository

Open the project folder

### - Run:

python hospital_gui.py
